### Flutter project

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/ThuDung213/flutter_lab.git
   cd micard
   ```
2. Install dependencies:
   ```sh
   flutter pub get
   ```
3. Run the app:
   ```sh
   flutter run
   ```
